# 🚀 Guia Rápido - Lista de Compras

## 📝 Primeiros Passos

### 1️⃣ Criar Categorias

- Digite o nome (ex: "Frutas", "Carnes", "Limpeza")
- Clique em "Adicionar"
- Repita para todas as categorias que precisa

### 2️⃣ Adicionar Itens

- Digite o nome do item
- Selecione a categoria no dropdown
- Clique em "Adicionar"

### 3️⃣ Definir Preços

- Clique no item ou no ícone de lápis
- Digite a quantidade e o valor
- O subtotal é calculado automaticamente
- Clique em "Salvar"

## 💡 Dicas de Uso

### Organização

- ✅ Crie categorias por seção do mercado
- ✅ Use nomes curtos e claros
- ✅ Agrupe itens similares

### Orçamento

1. Edite a categoria (ícone de lápis)
2. Defina um orçamento mensal
3. Acompanhe a barra de progresso:
   - 🟢 Verde = Está no controle
   - 🟡 Amarelo = Atenção! 80%+
   - 🔴 Vermelho = Orçamento estourado

### Compras

- ☑️ Marque itens conforme compra
- 🧹 Use "Limpar comprados" ao final
- 🔍 Use a busca para encontrar itens rapidamente

### Filtros e Busca

- 🔎 **Buscar**: Digite para filtrar em tempo real
- 📊 **Ordenar**:
  - Nome A-Z / Z-A
  - Menor / Maior preço
- 🧹 **Limpar**: Remove todos os itens marcados como comprados

## 📊 Análises

### Ver Gráficos

1. Clique em "Gráficos" no painel de resumo
2. **Gráfico Pizza**: Mostra quanto % gasta em cada categoria
3. **Gráfico Barras**: Compara valores entre categorias

### Exportar PDF

- Clique em "PDF" para baixar lista completa
- Inclui todas categorias, itens e totais
- Ótimo para imprimir e levar ao mercado

## 🎨 Personalização

### Tema Escuro

- Clique no ícone 🌙/☀️
- Alternates entre modo claro e escuro
- Salva sua preferência automaticamente

## 📱 Instalar no Celular

### Android/iOS

1. Abra o site no navegador
2. Menu (⋮) → "Instalar aplicativo"
3. Confirme a instalação
4. Use como app nativo! 📲

### Vantagens PWA

- ✨ Ícone na tela inicial
- ⚡ Carrega mais rápido
- 📶 Funciona offline (após 1º uso)
- 🔔 Recebe atualizações automáticas

## 🎯 Fluxo Ideal de Uso

```
1. Criar categorias
   ↓
2. Adicionar itens de cada categoria
   ↓
3. Definir orçamentos (opcional)
   ↓
4. Definir preços dos itens
   ↓
5. Ordenar por preço (ver o mais barato primeiro)
   ↓

   ↓

   ↓
```

## ⚡ Atalhos de Teclado

- **Enter** após digitar = Adiciona categoria/item
- **Tab** = Navega entre campos
- **Esc** = Fecha modais
- **Busca** = Filtra em tempo real

## 🔧 Troubleshooting

### Não consigo adicionar categoria/item

➡️ Verifique se preencheu o nome
➡️ Verifique a conexão com Firebase
➡️ Verifique as regras de segurança do Firestore

### Dados não sincronizam

➡️ Verifique sua conexão com internet
➡️ Recarregue a página (F5)
➡️ Limpe o cache do navegador

### PWA não instala

➡️ Use HTTPS (necessário para PWA)
➡️ Verifique se manifest.json está acessível
➡️ Use Chrome/Safari/Edge atualizado
